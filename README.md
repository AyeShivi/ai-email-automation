# AI Email Classification & Folder Automation System

AI-based Outlook email classification and folder automation system built during an internship at Jypra Group, Brisbane.

## Project Overview

Developed a standalone AI agent to automatically classify incoming Outlook emails and organise them into folders, reducing manual email triaging and improving team inbox efficiency.

## Business Problem

Teams spending time manually sorting high volumes of emails needed an automated solution to categorise and route messages instantly — freeing up time for higher-value work.

## Tech Stack

- **Language:** Python
- **ML / NLP:** Scikit-learn, text vectorisation
- **Integration:** Microsoft Graph API, Outlook
- **Deployment:** AWS Cloud
- **Storage:** SQLite, CSV

## Key Features

- Connects to Outlook via Microsoft Graph API
- Classifies emails using a trained machine learning model
- Automatically moves emails into categorised folders
- Tracks processed emails to avoid duplicates
- Supports custom category management

## Technical Skills Demonstrated

- **Machine Learning:** Model training, text classification, vectorisation
- **API Integration:** Microsoft Graph API, OAuth authentication
- **Cloud:** AWS deployment and hosting
- **Python:** Scripting, automation, file and database handling
- **Documentation:** Project report, process notes, test cases
